# HTTP/1.1

е°ұеңЁHTTP/1.0еҸ‘еёғзҡ„еҮ дёӘжңҲеҗҺпјҢеҸ‘еёғдәҶHTTPеҚҸи®®зҡ„з¬¬дёҖдёӘж ҮеҮҶеҢ–зүҲжң¬HTTP1.1гҖӮHTTP1.1ж¶ҲйҷӨдәҶеӨ§йҮҸжӯ§д№үеҶ…е®№е№¶ж–°еўһдәҶеҫҲеӨҡж”№иҝӣпјҡ

  • жҢҒд№…иҝһжҺҘпјҡTCPиҝһжҺҘй»ҳи®ӨдёҚе…ій—ӯпјҢиҠӮзңҒдәҶеӨҡж¬Ўжү“ејҖTCPиҝһжҺҘзҡ„ж—¶й—ҙгҖӮ
  • з®ЎйҒ“жңәеҲ¶Pipelineпјҡе…Ғи®ёеңЁз¬¬дёҖдёӘеә”зӯ”иў«е®Ңе…ЁеҸ‘йҖҒд№ӢеүҚе°ұеҸ‘йҖҒз¬¬дәҢдёӘиҜ·жұӮпјҢд»ҘйҷҚдҪҺйҖҡдҝЎе»¶иҝҹгҖӮ
  • ж”ҜжҢҒе“Қеә”еҲҶеқ—пјҡдә§з”ҹдёҖеқ—ж•°жҚ®еҲҷеҸ‘йҖҒдёҖеқ—пјҢеҚіжөҒжЁЎејҸ(stream)пјҢеңЁиҜ·жұӮжҲ–еӣһеә”зҡ„еӨҙдҝЎжҒҜдёӯжңүTransfer-Encodingеӯ—ж®өпјҢиЎЁжҳҺеӣһеә”з”ұж•°жҚ®еқ—з»„жҲҗ
  • ж–°еўһHostеӯ—ж®өпјҢиғҪеӨҹдҪҝдёҚеҗҢеҹҹеҗҚй…ҚзҪ®еңЁеҗҢдёҖдёӘ IP ең°еқҖзҡ„жңҚеҠЎеҷЁдёҠгҖӮ
  • еј•е…ҘдәҶиӢҘе№Ізј“еӯҳжңәеҲ¶

# HTTPзј“еӯҳжңәеҲ¶

HTTPзј“еӯҳжңүејәзј“еӯҳе’ҢеҚҸе•Ҷзј“еӯҳгҖӮејәзј“еӯҳе’ҢеҚҸе•Ҷзј“еӯҳеңЁзј“еӯҳе‘Ҫдёӯж—¶йғҪдјҡзӣҙжҺҘдҪҝз”Ёжң¬ең°зҡ„зј“еӯҳеүҜжң¬пјҢеҢәеҲ«еңЁдәҺеҚҸе•Ҷзј“еӯҳдјҡеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒдёҖж¬ЎиҜ·жұӮжқҘзЎ®и®ӨжҳҜеҗҰдҪҝз”Ёзј“еӯҳгҖӮ

еҪ“жөҸи§ҲеҷЁжңӘе‘Ҫдёӯејәзј“еӯҳж—¶пјҢеҲҷдјҡе‘ҪдёӯеҚҸе•Ҷзј“еӯҳ

# ејәзј“еӯҳ

зј“еӯҳзӯ–з•ҘпјҡеҰӮжһңзј“еӯҳиө„жәҗжңүж•ҲпјҢеҲҷзӣҙжҺҘдҪҝз”Ёзј“еӯҳиө„жәҗпјҢдёҚеҶҚеҗ‘жңҚеҠЎеҷЁйҮҚж–°иҜ·жұӮ и®ҫзҪ®ж–№ејҸпјҡHTTPж ҮеӨҙеӯ—ж®ө

1.ExpiresпјҡжҢҮе®ҡиө„жәҗзҡ„иҝҮжңҹж—¶й—ҙпјҢжҳҜдёҖдёӘз»қеҜ№ж—¶й—ҙгҖӮеҪ“жң¬ең°ж—¶й—ҙе°ҸдәҺи®ҫзҪ®зҡ„ж—¶й—ҙпјҢеҲҷдҪҝз”Ёзј“еӯҳгҖӮ жіЁж„Ҹпјҡж— ж•Ҳзҡ„ж—ҘжңҹпјҢжҜ”еҰӮ 0пјҢд»ЈиЎЁзқҖиҝҮеҺ»зҡ„ж—ҘжңҹпјҢеҚіиҜҘиө„жәҗе·Із»ҸиҝҮжңҹгҖӮеҰӮжһңеңЁCache-Controlе“Қеә”еӨҙи®ҫзҪ®дәҶ "max-age" жҲ–иҖ… "s-max-age" жҢҮд»ӨпјҢйӮЈд№Ҳ Expires еӨҙдјҡиў«еҝҪз•ҘгҖӮ

2.cache-controlпјҡи®ҫзҪ®зј“еӯҳжңәеҲ¶пјҢеёёз”Ёеӯ—ж®өжңү

[1]no-cacheпјҡеңЁеҸ‘еёғзј“еӯҳеүҜжң¬д№ӢеүҚпјҢејәеҲ¶иҝӣиЎҢеҚҸе•Ҷзј“еӯҳйӘҢиҜҒ

[2]no-store: дёҚдҪҝз”Ёд»»дҪ•зј“еӯҳ

[3]max-ageпјҡзӣёеҜ№иҝҮжңҹж—¶й—ҙпјҢеӨҡе°‘з§’еҶ…жңүж•Ҳ

# еҚҸе•Ҷзј“еӯҳ

зј“еӯҳзӯ–з•ҘпјҡеңЁдҪҝз”Ёзј“еӯҳж—¶дјҡеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒиҜ·жұӮпјҢй—®иҜўиө„жәҗжҳҜеҗҰеҸ‘з”ҹжӣҙж”№пјҢиӢҘжңӘеҸ‘з”ҹжӣҙж”№еҲҷиҝ”еӣһ304зҠ¶жҖҒпјҢе®ўжҲ·з«ҜзӣҙжҺҘдҪҝз”Ёзј“еӯҳпјҢеҰӮжһңиө„жәҗеҸ‘з”ҹж”№еҸҳпјҢеҲҷдҪҝз”ЁжңҚеҠЎеҷЁиҝ”еӣһиө„жәҗ и®ҫзҪ®ж–№ејҸпјҡHTTPж ҮеӨҙеӯ—ж®ө

1.Last-ModifiedпјҡиЎЁзӨәж–Үд»¶жңҖеҗҺиў«дҝ®ж”№зҡ„ж—ҘжңҹпјҢе®ўжҲ·з«ҜеҶҚж¬ЎиҜ·жұӮж—¶пјҢдјҡйҖҡиҝҮIf-Modified-Sinceеӯ—ж®өе°ҶLast-ModifiedеҖјеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁпјҢиҜўй—®жңҚеҠЎеҷЁеңЁиҝҷдёӘж—ҘжңҹеҗҺжҳҜеҗҰжңүжӣҙж–°гҖӮ

2.Etag:жңҚеҠЎз«ҜйҖҡиҝҮжҹҗз§Қз®—жі•и®Ўз®—еҮәиҜҘиө„жәҗзҡ„е”ҜдёҖж ҮиҜҶз¬ҰгҖӮIf-None-Matchдјҡе°ҶEtagеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁпјҢиҜўй—®иҜҘиө„жәҗжҳҜеҗҰжңүеҸҳеҠЁпјҢжңүеҸҳеҠЁзҡ„иҜқе°Ҷж–°зҡ„иө„жәҗиҝ”еӣһгҖӮEtagдјҳе…Ҳзә§жҜ”Last-Modifiedй«ҳгҖӮ

# HTTP/2.0

2015е№ҙ5жңҲжӯЈејҸеҸ‘еёғHTTP/2еҚҸи®®и§„иҢғпјҢиҜҘеҚҸи®®дҪҝз”ЁеӨҡи·ҜеӨҚз”ЁжңәеҲ¶пјҢе®һзҺ°дёҖдёӘеҹҹеҗҚеҸӘдҪҝз”ЁдёҖдёӘTCPй•ҝиҝһжҺҘпјҢе№¶ж¶ҲйҷӨдәҶйҳҹеӨҙйҳ»еЎһй—®йўҳгҖӮеӨҡи·ҜеӨҚз”ЁжҠҖжңҜиғҪе……еҲҶеҲ©з”ЁеёҰе®ҪпјҢжңҖеӨ§йҷҗеәҰи§„йҒҝдәҶTCPж…ўеҗҜеҠЁжүҖеёҰжқҘзҡ„й—®йўҳпјҢдҪҝеҫ—йЎөйқўиө„жәҗзҡ„дј иҫ“йҖҹеәҰеҫ—еҲ°дәҶеӨ§е№…жҸҗеҚҮгҖӮ дҪҝз”ЁHTTP/2иғҪеёҰжқҘ20% ~ 60%зҡ„ж•ҲзҺҮжҸҗеҚҮгҖӮ HTTP/2ж·»еҠ дәҶдёҖдёӘдәҢиҝӣеҲ¶еҲҶеё§еұӮпјҢе°Ҷз»ҸиҝҮзҡ„иҜ·жұӮиҪ¬жҚўдёәдёҖдёӘдёӘеёҰжңүиҜ·жұӮIDзј–еҸ·зҡ„её§пјҢжңҚеҠЎеҷЁжҺҘж”¶еҲ°жүҖжңүеё§д№ӢеҗҺпјҢе°ҶжүҖжңүзӣёеҗҢIDзҡ„её§еҗҲе№¶дёәдёҖжқЎе®Ңж•ҙзҡ„иҜ·жұӮдҝЎжҒҜпјҢеӨ„зҗҶе®ҢиҜ·жұӮеҗҺпјҢе°Ҷе“Қеә”д№ҹеҗҢж ·з”ЁдәҢиҝӣеҲ¶еҲҶеё§еұӮиҪ¬жҚўдёәдёҖдёӘдёӘеёҰжңүиҜ·жұӮIDзј–еҸ·зҡ„её§пјҢжөҸи§ҲеҷЁжҺҘж”¶еҲ°е“Қеә”её§еҗҺж №жҚ®IDзј–еҸ·е°Ҷж•°жҚ®жҸҗдәӨз»ҷеҜ№еә”зҡ„иҜ·жұӮгҖӮ

HTTP/2 жҳҜдәҢиҝӣеҲ¶еҚҸи®®иҖҢдёҚжҳҜж–Үжң¬еҚҸи®®гҖӮ иҝҷжҳҜдёҖдёӘеӨҡи·ҜеӨҚз”ЁеҚҸи®®гҖӮе№¶иЎҢзҡ„иҜ·жұӮиғҪеңЁеҗҢдёҖдёӘй“ҫжҺҘдёӯеӨ„зҗҶпјҢз§»йҷӨдәҶ HTTP/1.x дёӯйЎәеәҸе’Ңйҳ»еЎһзҡ„зәҰжқҹ еҺӢзј©дәҶж ҮеӨҙ

  • дәҢиҝӣеҲ¶еҲҶеё§пјҡHTTP/2 зҡ„жүҖжңүеё§йғҪйҮҮз”ЁдәҢиҝӣеҲ¶зј–з Ғ
  • еӨҡи·ҜеӨҚз”Ё (Multiplexing)
  • иҜ·жұӮдјҳе…Ҳзә§
  • header еҺӢзј©
  • жңҚеҠЎз«ҜжҺЁйҖҒ

дәҢиҝӣеҲ¶еҲҶеё§пјҡHTTP/2 зҡ„жүҖжңүеё§йғҪйҮҮз”ЁдәҢиҝӣеҲ¶зј–з Ғ е…ҲзҗҶи§ЈеҮ дёӘжҰӮеҝөпјҡ

её§пјҡе®ўжҲ·з«ҜдёҺжңҚеҠЎеҷЁйҖҡиҝҮдәӨжҚўеё§жқҘйҖҡдҝЎпјҢеё§жҳҜеҹәдәҺиҝҷдёӘж–°еҚҸи®®йҖҡдҝЎзҡ„жңҖе°ҸеҚ•дҪҚгҖӮ

ж¶ҲжҒҜпјҡжҳҜжҢҮйҖ»иҫ‘дёҠзҡ„ HTTP ж¶ҲжҒҜпјҢжҜ”еҰӮиҜ·жұӮгҖҒе“Қеә”зӯүпјҢз”ұдёҖжҲ–еӨҡдёӘеё§з»„жҲҗгҖӮ

жөҒпјҡжөҒжҳҜиҝһжҺҘдёӯзҡ„дёҖдёӘиҷҡжӢҹдҝЎйҒ“пјҢеҸҜд»ҘжүҝиҪҪеҸҢеҗ‘зҡ„ж¶ҲжҒҜпјӣжҜҸдёӘжөҒйғҪжңүдёҖдёӘе”ҜдёҖзҡ„ж•ҙж•°ж ҮиҜҶз¬ҰпјҲ1гҖҒ2вҖҰNпјүпјӣ

HTTP/2 йҮҮз”ЁдәҢиҝӣеҲ¶ж јејҸдј иҫ“ж•°жҚ®пјҢиҖҢйқһ HTTP 1.x зҡ„ж–Үжң¬ж јејҸпјҢдәҢиҝӣеҲ¶еҚҸи®®и§Јжһҗиө·жқҘжӣҙй«ҳж•ҲгҖӮ HTTP / 1 зҡ„иҜ·жұӮе’Ңе“Қеә”жҠҘж–ҮпјҢйғҪжҳҜз”ұиө·е§ӢиЎҢпјҢйҰ–йғЁе’Ңе®һдҪ“жӯЈж–ҮпјҲеҸҜйҖүпјүз»„жҲҗпјҢеҗ„йғЁеҲҶд№Ӣй—ҙд»Ҙж–Үжң¬жҚўиЎҢз¬ҰеҲҶйҡ”гҖӮHTTP/2 е°ҶиҜ·жұӮе’Ңе“Қеә”ж•°жҚ®еҲҶеүІдёәжӣҙе°Ҹзҡ„её§пјҢе№¶дё”е®ғд»¬йҮҮз”ЁдәҢиҝӣеҲ¶зј–з ҒгҖӮ

# её§гҖҒжөҒгҖҒж¶ҲжҒҜзҡ„е…ізі»

жҜҸдёӘж•°жҚ®жөҒйғҪд»Ҙж¶ҲжҒҜзҡ„еҪўејҸеҸ‘йҖҒпјҢиҖҢж¶ҲжҒҜеҸҲз”ұдёҖдёӘжҲ–еӨҡдёӘеё§з»„жҲҗгҖӮ её§жҳҜжөҒдёӯзҡ„ж•°жҚ®еҚ•дҪҚгҖӮдёҖдёӘж•°жҚ®жҠҘзҡ„ header её§еҸҜд»ҘеҲҶжҲҗеӨҡдёӘ header её§пјҢdata её§еҸҜд»ҘеҲҶжҲҗеӨҡдёӘ data её§гҖӮ

# еӨҡи·ҜеӨҚз”Ё (Multiplexing)

еӨҡи·ҜеӨҚз”Ёе…Ғи®ёеҗҢж—¶йҖҡиҝҮеҚ•дёҖзҡ„ HTTP/2 иҝһжҺҘеҸ‘иө·еӨҡйҮҚзҡ„иҜ·жұӮ-е“Қеә”ж¶ҲжҒҜгҖӮеҚіиҝһжҺҘе…ұдә«пјҢеҚіжҜҸдёҖдёӘ request йғҪжҳҜжҳҜз”ЁдҪңиҝһжҺҘе…ұдә«жңәеҲ¶зҡ„гҖӮдёҖдёӘ request еҜ№еә”дёҖдёӘ idпјҢиҝҷж ·дёҖдёӘиҝһжҺҘдёҠеҸҜд»ҘжңүеӨҡдёӘ requestпјҢжҜҸдёӘиҝһжҺҘзҡ„ request еҸҜд»ҘйҡҸжңәзҡ„ж··жқӮеңЁдёҖиө·пјҢжҺҘж”¶ж–№еҸҜд»Ҙж №жҚ® request зҡ„ id е°Ҷ request еҶҚеҪ’еұһеҲ°еҗ„иҮӘдёҚеҗҢзҡ„жңҚеҠЎз«ҜиҜ·жұӮйҮҢйқўгҖӮ

# иҜ·жұӮдјҳе…Ҳзә§

  • жҠҠ HTTP ж¶ҲжҒҜеҲҶи§ЈдёәеҫҲеӨҡзӢ¬з«Ӣзҡ„её§д№ӢеҗҺпјҢе°ұеҸҜд»ҘйҖҡиҝҮдјҳеҢ–иҝҷдәӣеё§зҡ„дәӨй”ҷе’Ңдј иҫ“йЎәеәҸпјҢжҜҸдёӘжөҒйғҪеҸҜд»ҘеёҰжңүдёҖдёӘ 31 жҜ”зү№зҡ„дјҳе…ҲеҖјпјҡ0 иЎЁзӨәжңҖй«ҳдјҳе…Ҳзә§пјӣ2 зҡ„ 31 ж¬Ўж–№-1 иЎЁзӨәжңҖдҪҺдјҳе…Ҳзә§гҖӮ
  • жңҚеҠЎеҷЁеҸҜд»Ҙж №жҚ®жөҒзҡ„дјҳе…Ҳзә§пјҢжҺ§еҲ¶иө„жәҗеҲҶй…ҚпјҲCPUгҖҒеҶ…еӯҳгҖҒеёҰе®ҪпјүпјҢиҖҢеңЁе“Қеә”ж•°жҚ®еҮҶеӨҮеҘҪд№ӢеҗҺпјҢдјҳе…Ҳе°ҶжңҖй«ҳдјҳе…Ҳзә§зҡ„её§еҸ‘йҖҒз»ҷе®ўжҲ·з«ҜгҖӮ
  • HTTP 2.0 дёҖдёҫи§ЈеҶідәҶжүҖжңүиҝҷдәӣдҪҺж•Ҳзҡ„й—®йўҳпјҡжөҸи§ҲеҷЁеҸҜд»ҘеңЁеҸ‘зҺ°иө„жәҗж—¶з«ӢеҚіеҲҶжҙҫиҜ·жұӮпјҢжҢҮе®ҡжҜҸдёӘжөҒзҡ„дјҳе…Ҳзә§пјҢи®©жңҚеҠЎеҷЁеҶіе®ҡжңҖдјҳзҡ„е“Қеә”ж¬ЎеәҸгҖӮиҝҷж ·иҜ·жұӮе°ұдёҚеҝ…жҺ’йҳҹдәҶпјҢж—ўиҠӮзңҒдәҶж—¶й—ҙпјҢд№ҹжңҖеӨ§йҷҗеәҰең°еҲ©з”ЁдәҶжҜҸдёӘиҝһжҺҘгҖӮ

# header еҺӢзј©

HTTP1.x зҡ„ header еёҰжңүеӨ§йҮҸдҝЎжҒҜпјҢиҖҢдё”жҜҸж¬ЎйғҪиҰҒйҮҚеӨҚеҸ‘йҖҒпјҢHTTP/2 дҪҝз”Ё encoder жқҘеҮҸе°‘йңҖиҰҒдј иҫ“зҡ„ header еӨ§е°ҸпјҢйҖҡи®ҜеҸҢж–№еҗ„иҮӘcache дёҖд»Ҫ header fields иЎЁпјҢж—ўйҒҝе…ҚдәҶйҮҚеӨҚ header зҡ„дј иҫ“пјҢеҸҲеҮҸе°ҸдәҶйңҖиҰҒдј иҫ“зҡ„еӨ§е°ҸгҖӮ дёәдәҶеҮҸе°‘иҝҷеқ—зҡ„иө„жәҗж¶ҲиҖ—е№¶жҸҗеҚҮжҖ§иғҪпјҢ HTTP/2 еҜ№иҝҷдәӣйҰ–йғЁйҮҮеҸ–дәҶеҺӢзј©зӯ–з•Ҙпјҡ

  • HTTP/2 еңЁе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«ҜдҪҝз”ЁвҖңйҰ–йғЁиЎЁвҖқжқҘи·ҹиёӘе’ҢеӯҳеӮЁд№ӢеүҚеҸ‘йҖҒзҡ„й”®пјҚеҖјеҜ№пјҢдёҚеҶҚйҮҚеӨҚеҸ‘йҖҒ header
  • йҰ–йғЁиЎЁеңЁ HTTP/2 зҡ„иҝһжҺҘеӯҳз»ӯжңҹеҶ…е§Ӣз»ҲеӯҳеңЁпјҢз”ұе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁе…ұеҗҢжёҗиҝӣең°жӣҙж–°;
  • жҜҸдёӘж–°зҡ„йҰ–йғЁй”®пјҚеҖјеҜ№иҰҒд№Ҳиў«иҝҪеҠ еҲ°еҪ“еүҚиЎЁзҡ„жң«е°ҫпјҢиҰҒд№ҲжӣҝжҚўиЎЁдёӯд№ӢеүҚзҡ„еҖј

# жңҚеҠЎз«ҜжҺЁйҖҒ

Server Push еҚіжңҚеҠЎз«ҜиғҪйҖҡиҝҮ push зҡ„ж–№ејҸе°Ҷе®ўжҲ·з«ҜйңҖиҰҒзҡ„еҶ…е®№йў„е…ҲжҺЁйҖҒиҝҮеҺ»пјҢд№ҹеҸ«вҖңcache pushвҖқгҖӮ жңҚеҠЎеҷЁеҸҜд»ҘеҜ№дёҖдёӘе®ўжҲ·з«ҜиҜ·жұӮеҸ‘йҖҒеӨҡдёӘе“Қеә”гҖӮжңҚеҠЎеҷЁеҗ‘е®ўжҲ·з«ҜжҺЁйҖҒиө„жәҗж— йңҖе®ўжҲ·з«ҜжҳҺзЎ®ең°иҜ·жұӮпјҢжңҚеҠЎз«ҜеҸҜд»ҘжҸҗеүҚз»ҷе®ўжҲ·з«ҜжҺЁйҖҒеҝ…иҰҒзҡ„иө„жәҗпјҢиҝҷж ·еҸҜд»ҘеҮҸе°‘иҜ·жұӮе»¶иҝҹж—¶й—ҙпјҢдҫӢеҰӮжңҚеҠЎз«ҜеҸҜд»Ҙдё»еҠЁжҠҠ JS е’Ң CSS ж–Үд»¶жҺЁйҖҒз»ҷе®ўжҲ·з«ҜпјҢиҖҢдёҚжҳҜзӯүеҲ° HTML и§ЈжһҗеҲ°иө„жәҗж—¶еҸ‘йҖҒиҜ·жұӮ
